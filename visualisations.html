Visual depiction of is usually the fastest way to grasping a physical problem in question. Complication may arise when there are more dimensions of data than we can visualise. For this reason, it is imperative to provide rich visualisation containing as much information as possible, yet simple enough so one would not get overwhelmed with uncomprehensable data. Geodesic motion in 4- and 5-dimensional spacetimes is therefore visualised in interactive graphical interface using plotly library. Geodesics are visualised as projection to 3-dimensional space (one dimension is neglected). Each computed point in graph contains propper time on geodesic (or affine parameter in case of null geodesics), where the value of 0 corresponds to initial position. In case of impulsive spacetimes visualised using refraction equations, value of 0 is at the exact moment of crossing the wavefront.
